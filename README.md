# Portfolio Setup Task

## Overview
This repository documents the setup and tools installation process as part of the assigned portfolio task. The goal was to set up a development environment using Cursor IDE, integrate AI coding tools, and connect everything with GitHub for version control.

---

## Tools Installed

- Cursor IDE
- Claude Code extension (inside Cursor)
- Codex extension (inside Cursor)
- Git
- GitHub account and repository setup

---

## Steps Completed

### 1. Cursor IDE Installation
Installed Cursor IDE from the official website and completed the initial setup successfully.

### 2. Claude Code Integration
Installed the Claude Code extension from the Cursor Extensions marketplace and logged into the service.

### 3. Codex Integration
Installed the Codex extension inside Cursor and completed authentication.

### 4. GitHub Setup
Created a GitHub account and initialized a new public repository for this project.

### 5. Repository Connection
Cloned the GitHub repository locally and opened it inside Cursor IDE for development and tracking.

### 6. README Documentation
Created this README.md file to document all steps, tools used, and setup experience.

### 7. Version Control
Committed the project files and pushed them to the GitHub repository.

---

## Challenges & Learnings

- Initial setup of extensions required account authentication and permissions.
- Git configuration and pushing to GitHub required careful setup of repository connection.
- Learned how to integrate multiple AI coding tools inside a single development environment.
- Understood how Cursor streamlines AI-assisted development workflows.

---

## Outcome

Successfully set up a complete development environment with AI-assisted coding tools and connected it with GitHub for version control. This setup is now ready for building and showcasing future projects.
